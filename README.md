# SONIVERSE

Soniverse is a marketplace for audio sound effects creators where they can monetize their work using the Tezos blockchain network.

To run the project:

cd {project}
npm install
npm start

Temple - Tezos Wallet (https://chrome.google.com/webstore/detail/temple-tezos-wallet/ookjlbkiijinhpmnjffcofjonbfbgaoc) 
can be utilized to connect your wallet and start minting or buy other audio samples.

At the moment the smart contracts are deployed on JAKARTANET.
